## 2026-05-29 v2.2 — Flask 独立管理后台（方案B）

```

         Nginx (80)
         ↙          ↘
  /admin/*            / 
    ↓                  ↓

(独立服务)        (聊天网站)
```

**变更内容：**
- 新建 `/root/web_ai/admin_backend/` 目录
- `app.py` — Flask 3.1.3 应用，含路由：
  - `/admin/login` — 管理员登录
  - `/admin/` — 总览面板（用户数、消息量、7天趋势、评分统计）
  - `/admin/users` — 用户管理（查看/删除/设为管理员）
  - `/admin/chats` — 聊天记录查询（按用户筛选）
- 4 个 Jinja2 模板（Bootstrap 5.3 样式）
- Nginx 配置更新：`/admin/` 路由到 Flask :5000
- Flask 加入 systemd 服务 `flask-admin.service`（随系统自启）

**后台 vs 方案A区别：**
| 特性 | 方案A (Streamlit) | 方案B (Flask) |
|:----|:-----------------:|:-------------:|
| 响应速度 | 稍慢（全量rerun） | 快速 |
| 样式 | 原生 Streamlit | Bootstrap |
| 独立端口 | 共用 | 独立 :5000 |
| 配置复杂度 | 低 | 中 |
| 扩展性 | 受限于 Streamlit | 自由 |

**效果：** 两个后台并存，均可使用，由你自己决定留哪个。

## 2026-05-29 v2.3 — 数据备份 + API 限流

### 🗄️ 新增：MySQL 自动备份

**配置：**
- 备份脚本：`/root/backup_mysql.sh`
- 执行时间：每天凌晨 3:00（crontab）
- 存储路径：`/backup/web_ai_YYYYMMDD_HHMMSS.sql.gz`

**影响文件：**
- `app_qwen.py` — 用户输入前先检查限流，超限直接阻止
- `models/admin.py` — 新增 `check_rate_limit()` / `get_rate_limit_stats()`

**效果：** 防止刷 API 烧钱，测试账号不小心狂点也不会扣费。

### 🐛 顺手修复

## 2026-06-01 v3.0 — Docker 化部署

### 🐳 架构变更：裸机 → Docker Compose

**之前（裸机）：**
```
Nginx(host) → Streamlit(host) + Flask(host) + MySQL(host)
```

**之后（Docker）：**
```
Nginx(host) → Streamlit(容器) + Flask(容器) + MySQL(容器)
              docker-compose 管理，重启自启
```

**新建文件：**
- `docker-compose.yml` — 编排 3 个服务
- `yijian_AI/Dockerfile` — Streamlit 镜像（592MB）
- `admin_backend/Dockerfile` — Flask 镜像（141MB）

**变更内容：**
- MySQL 迁移至 Docker 容器（挂载 volume 持久化数据）
- Streamlit 容器化（清华 pip 源加速构建）
- Flask 容器化（监听 0.0.0.0 修复端口转发）
- Nginx 保留在宿主机（避免中断，后续也可容器化）
- 备份脚本改为从 Docker MySQL 备份
- Host MySQL 服务已停止并禁用自启
- Docker 容器设为 `restart: unless-stopped`

**修复的坑：**
- ❌ Tencent PyPI 镜像 166B/s → 清华源 198KB/s ✅
- ❌ Flask 监听 127.0.0.1 导致 Docker 端口转发失败 → 改为 0.0.0.0 ✅
- ❌ 多个 docker build 进程冲突 → 清理后单次构建 ✅

**当前架构：**
```
访客 → http://119.45.174.234
            ↓
      Nginx :80 (宿主机)
      ↙               ↘
/admin/*              /
  ↓                    ↓
Flask :5000     Streamlit :8501
(Docker)         (Docker)
  ↓                    ↓
     MySQL :3306 (Docker, 数据持久化)
          ↓
    每天3点自动备份 → /backup/
```
